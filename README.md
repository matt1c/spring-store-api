# Spring Store API

REST API для интернет-магазина, построенный на Spring Boot. Поддерживает аутентификацию через JWT, управление пользователями с ролями, каталог товаров и систему заказов с резервированием остатков.

## Возможности

- **Аутентификация** - регистрация и вход через JWT-токены
- **Роли** - ROLE_USER, ROLE_ADMIN, ROLE_MANAGER с разграничением доступа
- **Товары** - CRUD-операции с пагинацией и сортировкой
- **Заказы** - создание заказов с автоматическим резервированием остатков на складе
- **Админ-панель** - управление пользователями (бан/разбан, назначение ролей)

## Стек

- Spring Security + JWT
- Spring Data JPA

## Система заказов

При создании заказа остаток товара **резервируется сразу** — количество списывается со склада в момент оформления. Это гарантирует, что один и тот же товар не будет продан дважды.

- Если остатка не хватает — возвращается ошибка `400 Bad Request`
- При отмене заказа (`CANCELLED`) — зарезервированный остаток возвращается на склад
- При доставке (`DELIVERED`) — остаток уже списан, дополнительных действий не требуется

Каждая позиция в заказе хранит цену товара на момент оформления (`priceAtOrder`), поэтому изменение цены в каталоге не влияет на уже созданные заказы.

## Запуск

```bash
# Локально
./mvnw spring-boot:run

# Тесты
./mvnw test
```
